Um projeto chamado "Eventos DOM", é um arquivo HTML com JavaScript que permite que quando você clicar em um bloco (uma div), aconteça um evento, por exemplo o site interagir com o usuario, que aconteça algo depois do usuario clicar.. uma mensagem, um alerta, uma foto, alguma resposta da pagina.
